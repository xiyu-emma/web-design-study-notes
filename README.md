# 📚 網頁設計課程筆記複習網站

一個簡單且交互式的網頁設計課程筆記複習網站，基於 Notion 課堂筆記製作，使用純 HTML 和 CSS 實現。

## ✨ 功能特色

- 📅 **課程概覽**: 完整的8週課程內容規劃
- 📝 **HTML基礎**: 圖片、超連結、清單等標籤使用
- 🎨 **CSS樣式**: 選取器、優先級、樣式引入方式
- 📀 **版面配置**: 區塊置中、浮動排版、定位技術
- ⚡ **動態效果**: Transition、Transform、互動效果
- 🎯 **實作範例**: 房型介紹頁面等實際應用

## 🚀 快速開始

### 線上瀏覽

直接點擊這裡在線瀏覽：
**[https://xiyu-emma.github.io/web-design-study-notes/](https://xiyu-emma.github.io/web-design-study-notes/)**

### 本地端運行

1. 複製專案：
   ```bash
   git clone https://github.com/xiyu-emma/web-design-study-notes.git
   ```

2. 進入專案目錄：
   ```bash
   cd web-design-study-notes
   ```

3. 用瀏覽器開啟 `index.html` 檔案即可。

## 📚 內容結構

### 課程概覽
- 第1-2週：HTML基礎（文字、圖片、清單、超連結）
- 第3週：進階HTML應用（複雜超連結、圖文混排）
- 第4-5週：CSS基礎（樣式表基礎與串接）
- 第6週：版面配置（Div定位）
- 第7週：多媒體整合（背景影像、聲音、影片）
- 第8週：動態效果（CSS動態效果、互動元件）

### 技術詳細內容
- **HTML標籤使用**：`<img>`、`<a>`、`<ul>`、`<li>` 等
- **CSS選取器**：標籤、Class、ID、後代、子代選取器
- **版面配置**：水平置中、浮動排版、定位技術
- **動態效果**：Transition、Transform、關鍵影格動畫

## 🛠️ 技術堆疊

- **HTML5**: 語意化標籤和結構
- **CSS3**: 現代樣式設計和動畫效果
- **響應式設計**: 支援手機和平板裝置
- **JavaScript**: 基本交互功能（平滑滾動、導航高亮）

## 🌐 網站特色

### 設計理念
- 簡單且清晰的介面設計
- 色彩調和且舸目的視覺效果
- 優雅的動畫轉場
- 直覺的導航結構

### 交互功能
- 平滑滾動到特定節區
- 導航列按鈕高亮顯示
- Hover 效果示範
- 變形效果交互展示

### 響應式設計
- 支援桌面、平板和手機裝置
- 彈性網格布局
- 自適應字體和間距
