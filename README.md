# 📚 網頁設計課程筆記複習網站

一個簡單且清晰的網頁設計課程筆記複習網站，基於 Notion 課堂筆記製作，使用純 HTML 和 CSS 實現。

## ✨ 功能特色

- 🏠 **首頁總覽**: 清晰的課程結構和學習路徑
- 📝 **分頁設計**: 每個主題獨立成頁，方便專注學習
- 🎨 **簡潔風格**: 清晰明瞭的設計，無干擾學習
- 📱 **響應式設計**: 支援桌面、平板、手機裝置
- ⚡ **無JavaScript**: 純HTML/CSS實現，輕量且快速

## 🚀 快速開始

### 線上瀏覽

直接點擊這裡在線瀏覽：
**[https://xiyu-emma.github.io/web-design-study-notes/](https://xiyu-emma.github.io/web-design-study-notes/)**

### 本地端運行

1. 複製專案：
   ```bash
   git clone https://github.com/xiyu-emma/web-design-study-notes.git
   ```

2. 進入專案目錄：
   ```bash
   cd web-design-study-notes
   ```

3. 用瀏覽器開啟 `index.html` 檔案即可。

## 📚 網站結構

### 頁面列表
- **首頁** (`index.html`): 課程總覽和學習路徑
- **HTML基礎** (`html-basics.html`): HTML標籤和屬性使用
- **CSS基礎** (`css-basics.html`): CSS選取器和樣式語法
- **版面配置** (`layout.html`): 浮動、定位和布局技術
- **動態效果** (`effects.html`): CSS動畫和轉場效果
- **實作範例** (`examples.html`): 完整的項面實作範例

### 課程內容概覽

| 週次 | 主題 | 內容 |
|------|------|------|
| 1-2週 | HTML基礎 | 文字排版、圖片嵌入、清單、超連結 |
| 3週 | 進階HTML | 複雜超連結、圖文混排、表格布局 |
| 4-5週 | CSS基礎 | CSS語法、樣式表串接、基本樣式 |
| 6週 | 版面配置 | Div定位、版面配置技術 |
| 7週 | 多媒體整合 | 背景影像、聲音、影片 |
| 8週 | 動態效果 | CSS動畫、轉場、互動效果 |

## 🛠️ 技術堆疊

## 🎯 學習目標

這個網站適合：
- 🌱 **初學者**: 網頁設計新手循序漸進學習
- 🔍 **快速查找**: 快速查找HTML/CSS語法和用法
- 📚 **複習匯總**: 設計課程學生複習筆記
- 🔧 **實作參考**: 尋找實際專案的設計靈感

## 📚 學習路徑

### 基礎階段
1. **HTML基礎** - 掌握基本標籤和結構
2. **CSS基礎** - 學會樣式設計和選取器
3. **版面配置** - 掌握布局和定位技術
4. **動態效果** - 增加互動性和視覺吸引力
5. **實作練習** - 完成完整的網頁專案

### 進階學習
- HTML5 語意化標籤
- CSS3 進階特性
- Flexbox 和 Grid 布局
- 響應式設計原則
- CSS 框架 (Bootstrap, Tailwind)
- JavaScript 基礎互動

## 🌟 使用指南

1. **瀏覽學習**: 從首頁開始，按照順序點擊各頁面
2. **重點學習**: 直接跳到您需要的特定主題
3. **實作練習**: 在實作範例頁面查看完整代碼
4. **移動學習**: 在手機上隨時複習重點內容
