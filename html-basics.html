HTML基礎技術

1. 圖片嵌入 (<img>)

在網頁中嵌入圖片是最基本的技能之一。

<img src="images/logo.png" alt="網站Logo" width="960">

重要屬性說明：

  • src (Source)：指定圖片來源路徑
  • alt (Alternative Text)：圖片替代文字，有助於SEO和無障礙閱讀
  • width：設定圖片寬度（建議改用CSS）

2. 超連結 (<a>)

超連結是網頁之間連接的橋梁，可以連結到其他頁面或網站。

基本連結

<a href="test.html">連結文字</a>

圖片連結

<a href="test.html">
  <img src="images/logo.png" alt="網站Logo">
</a>

新視窗開啟

<a href="test.html" target="_blank">新視窗連結</a>

3. 清單與導航列

清單是組織內容的重要元素，常用於建立導航列。

<div align="center">
  <ul>
    <li style="display: inline;">
      <a href="#"><img src="images/nav/關於我們.png" alt="關於我們"></a>
    </li>
    <li style="display: inline;">
      <a href="#"><img src="images/nav/最新消息.png" alt="最新消息"></a>
    </li>
  </ul>
</div>

小訣竅：

使用 display: inline; 可以讓清單項目水平排列，適合製作導航列。

4. 表格版面配置

在CSS普及之前，表格是主要的版面配置工具。

<table border="1" align="center">
  <tr>
    <td align="center" valign="top">儲存格內容</td>
    <td>
      <!-- 巢狀表格 -->
      <table border="1">
        <tr><td>內部表格內容</td></tr>
      </table>
    </td>
  </tr>
</table>

注意：

現代網頁設計不建議使用表格做版面配置，應該使用CSS的Flexbox或Grid。
