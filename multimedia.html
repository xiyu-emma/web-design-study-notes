多媒體背景設定

1. 圖片背景設定

使用 CSS 的 background-image 屬性可以為網頁元素設定背景圖片。

基本語法

div {
  background-image: url('images/background.jpg');
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
}

背景圖片屬性說明：

  • background-image：指定背景圖片的路徑
  • background-size：控制圖片大小
    • cover：覆蓋整個區域，保持比例
    • contain：完整顯示圖片，保持比例
    • 100% 100%：拉伸至完全填滿
  • background-position：圖片位置（center、top、bottom等）
  • background-repeat：是否重複顯示

2. 全頁面背景圖片

為整個網頁設定背景圖片，創造更豐富的視覺體驗。

body {
  background-image: url('images/page-background.jpg');
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
  background-attachment: fixed; /* 固定背景 */
}

最佳實踐建議：

  • 選擇高解析度但檔案大小適中的圖片
  • 確保背景不會影響文字的可讀性
  • 考慮使用半透明遮罩提高對比度
  • 測試在不同螢幕尺寸下的效果

3. 影片背景設定

HTML5 的 <video> 標籤搭配 CSS 定位，可以創造動態的影片背景效果。

HTML 結構

<div class="video-background">
  <video autoplay muted loop>
    <source src="background-video.mp4" type="video/mp4">
    <source src="background-video.webm" type="video/webm">
    您的瀏覽器不支援影片標籤。
  </video>
  
  <div class="content-overlay">
    <h1>網站標題</h1>
    <p>在影片背景上的內容</p>
  </div>
</div>

CSS 樣式

.video-background {
  position: relative;
  width: 100%;
  height: 100vh; /* 全螢幕高度 */
  overflow: hidden;
}

.video-background video {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  object-fit: cover; /* 保持比例覆蓋 */
  z-index: -1; /* 置於背景層 */
}

.content-overlay {
  position: relative;
  z-index: 1;
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  height: 100%;
  color: #dfd9c7;
  text-align: center;
  background: rgba(0,0,0,0.3); /* 半透明遮罩 */
}

影片背景注意事項：

  • 檔案大小：影片檔案通常較大，注意載入速度
  • 自動播放：需要加上 muted 屬性才能自動播放
  • 多格式支援：提供多種格式確保相容性
  • 行動裝置：某些行動裝置可能不支援自動播放

4. 圖案背景 (CSS Pattern)

使用 CSS 漸層和重複功能，可以創造各種圖案背景效果。

/* 條紋圖案 */
.stripe-bg {
  background-image: repeating-linear-gradient(
    45deg,
    #e8e3d3,
    #e8e3d3 10px,
    #dfd9c7 10px,
    #dfd9c7 20px
  );
}

/* 點狀圖案 */
.dot-bg {
  background-image: radial-gradient(circle, #1a2b39 2px, transparent 2px);
  background-size: 20px 20px;
}

/* 格線圖案 */
.grid-bg {
  background-image: 
    linear-gradient(rgba(0,0,0,0.1) 1px, transparent 1px),
    linear-gradient(90deg, rgba(0,0,0,0.1) 1px, transparent 1px);
  background-size: 20px 20px;
}

5. 背景組合技巧

可以同時使用多種背景效果，創造層次豐富的視覺效果。

.complex-bg {
  /* 多重背景 */
  background: 
    linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)), /* 遮罩層 */
    url('overlay-pattern.png'),                          /* 圖案層 */
    url('main-background.jpg');                          /* 主背景 */
    
  background-size: 
    cover,        /* 遮罩層 */
    50px 50px,    /* 圖案層 */
    cover;        /* 主背景 */
    
  background-position: center, center, center;
  background-repeat: no-repeat, repeat, no-repeat;
}

背景層次順序：

  1. 最上層：遮罩或裝飾圖案
  2. 中間層：紋理或圖案
  3. 最底層：主要背景圖片

CSS 中先寫的背景會顯示在上層。

學習重點總結：

  • 圖片背景：使用 background-image 和相關屬性
  • 影片背景：結合 HTML5 video 標籤和 CSS 定位
  • 多重背景：創造豐富的視覺層次
  • 效能考量：注意檔案大小和載入速度
