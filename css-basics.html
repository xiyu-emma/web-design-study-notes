CSS樣式技術

1. CSS引入方式比較

CSS樣式可以通過三種不同的方式引入到HTML中。

方式 語法 優點 缺點 適用場合
行內樣式 <div style="width: 960px;"> 優先權高、測試方便 維護性極差 測試用途
嵌入式 <style> div { width: 960px; } </style> 單頁面集中管理 無法跨頁面共用 單頁應用
外部樣式表 <link rel="stylesheet" href="style.css"> 整站集中管理 需額外檔案 業界標準

2. CSS基本語法

CSS的基本結構由選取器和屬性組成。

/* 選取器 { 屬性: 值; } */
p {
  color: red;
  font-size: 16px;
}

3. 選取器類型

不同的選取器可以選擇不同的HTML元素。

標籤選取器

p {
  color: blue;
}

Class選取器

.box {
  width: 200px;
}

ID選取器

#header {
  height: 100px;
}

後代選取器

.box p {
  margin: 10px;
}

子代選取器

.box > p {
  padding: 5px;
}

4. 優先級順序

當多個樣式規則應用到同一元素時，瀏覽器會按照以下優先級順序決定使用哪個樣式。

優先級順序（由高到低）：

  1. !important 規則
  2. 行內樣式 (style 屬性)
  3. ID選取器 (#id)
  4. Class選取器 (.class)
  5. 標籤選取器 (tag)
  6. 瀏覽器預設樣式

5. 常用CSS屬性

文字樣式

p {
  font-size: 16px;        /* 字體大小 */
  font-weight: bold;      /* 字體粗細 */
  color: #333;            /* 文字顏色 */
  text-align: center;     /* 文字對齊 */
}

背景樣式

div {
  background-color: #f0f0f0;  /* 背景顏色 */
  background-image: url('bg.jpg'); /* 背景圖片 */
}

邊框與陰影

.box {
  border: 1px solid #ccc;     /* 邊框 */
  border-radius: 5px;         /* 圓角 */
  box-shadow: 0 2px 5px rgba(0,0,0,0.1); /* 陰影 */
}

學習建議：

  • 優先使用外部樣式表，便於維護和管理
  • 遵循命名約定，使用有意義的class名稱
  • 合理使用選取器，避免過度具體化
