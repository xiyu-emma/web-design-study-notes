版面配置技術

1. 區塊置中

水平置中是網頁設計中非常常見的需求。

#main {
  width: 800px;
  margin: 0 auto; /* 水平置中 */
}

置中方法說明：

  • margin: auto 用於區塊元素水平置中
  • text-align: center 用於文字置中
  • 區塊必須有固定寬度才能使用 margin: auto

2. 浮動排版

浮動是創建多欄布局的傳統方法。

.item {
  width: 180px;
  height: 180px;
  float: left;
  margin: 10px;
}

/* 清除浮動 */
.clear {
  clear: both;
}

浮動注意事項：

  • 浮動元素會脱離正常文檔流
  • 父元素可能失去高度
  • 必須適時清除浮動以避免布局問題

3. 定位技術

CSS定位允許您精確控制元素的位置。

相對定位

#container {
  position: relative; /* 定位參考點 */
}

絕對定位

#element {
  position: absolute;
  top: 100px;
  left: 150px;
}

定位簡化理解：

  • static：預設定位，遵循正常文檔流
  • relative：相對於原位置偏移
  • absolute：相對於最近的定位父元素
  • fixed：相對於瀏覽器視窗

4. 網格佈局範例

使用浮動建立簡單的網格系統。

#main {
  width: 500px;
  margin: auto;
  border: medium solid gray;
  border-radius: 20px;
}

.item {
  width: 180px;
  height: 180px;
  float: left;
  margin: 10px;
  border: thin solid green;
}

.item img {
  width: 100%;
}

5. 常見版面結構

網頁的基本結構通常包含以下部分：

網頁結構示意：

<div id="wrap">
  <div id="header">頁首區塊</div>
  <div id="nav">導航列</div>
  <div id="content">
    <div id="left">左欄內容</div>
    <div id="right">右欄內容</div>
  </div>
  <div id="footer">頁尾區塊</div>
</div>
/* 對應的CSS樣式 */
#wrap {
  width: 960px;
  margin: 0 auto;
}

#header {
  height: 100px;
  background: #1a2b39;
}

#nav {
  height: 50px;
  background: #7b2227;
}

#left {
  width: 200px;
  float: left;
  background: #cfc9b3;
}

#right {
  width: 760px;
  float: right;
  background: #f5f1e6;
}

#footer {
  clear: both;
  height: 80px;
  background: #5b6a5d;
}

版面配置技巧總結：

  • 水平置中：margin: auto 用於區塊，text-align: center 用於文字
  • 浮動排版：float: left/right 創建多欄佈局
  • 清除浮動：clear: both 恢復正常排版流
  • 絕對定位：配合 position: relative 的父容器使用
