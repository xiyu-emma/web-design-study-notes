CSS動態效果

1. 轉場效果 (Transition)

Transition 可以讓元素在狀態變化時產生平滑的過度效果。

.hover-demo {
  width: 200px;
  height: 60px;
  background: #3498db;
  transition: all 0.3s ease;
}

.hover-demo:hover {
  background: #e74c3c;
  transform: scale(1.1);
}

Transition 屬性說明：

  • transition-property：指定要加上過度效果的CSS屬性
  • transition-duration：轉場時間（如 0.3s）
  • transition-timing-function：動畫速度曲線
  • transition-delay：延遲時間

2. 2D變形 (Transform)

Transform 允許您旋轉、縮放、移動和傾斜元素。

常用變形函數

.transform-demo {
  transform: scale(1.2);           /* 縮放 */
  transform: rotate(45deg);        /* 旋轉 */
  transform: translate(50px, 50px); /* 移動 */
  transform: skew(20deg);          /* 傾斜 */
}

3. 組合變形

可以將多個變形函數組合使用。

.combined-transform {
  transform: rotate(45deg) scale(1.2) translate(20px, 20px);
  transform-origin: center center;
}

變形基準點 (Transform Origin)：

  • center center：中心點（預設）
  • top left：左上角
  • bottom right：右下角

4. 關鍵影格動畫 (@keyframes)

關鍵影格動畫可以創建更複雜的動畫效果。

@keyframes moveAround {
  0%   { transform: translate(0, 0); }
  25%  { transform: translate(100px, 0); }
  50%  { transform: translate(100px, 100px); }
  75%  { transform: translate(0, 100px); }
  100% { transform: translate(0, 0); }
}

.animated {
  animation: moveAround 2s linear infinite;
}

Animation 屬性說明：

  • animation-name：動畫名稱
  • animation-duration：動畫時間
  • animation-timing-function：速度曲線
  • animation-iteration-count：重複次數

5. 互動設計最佳實踐

建議使用場景：

  • Hover效果：按鈕、連結、圖片等互動元素
  • Transition：狀態變化時的平滑過度
  • Transform：微動畫效果
  • Keyframes：複雜的動畫序列

性能優化建議：

  • 優先使用 transform 和 opacity
  • 避免動畫過於複雜
  • 在行動裝置上測試效果
