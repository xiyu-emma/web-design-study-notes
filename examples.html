2. 完整網站版面(參考作業1和3)

經典的網站版面結構，包含頁首、導航、內容和頁尾。

頁首區域 (Header)
導航列 (Navigation)
左側選單
(Sidebar)

主內容區域 (Main Content)

這裡是網頁的主要內容區域，可以放置文字、圖片、表格等各種元素。

頁尾區域 (Footer)

HTML 結構：

<div id="wrap">
  <div id="header">
    <div id="logo">Logo區塊</div>
    <div id="menubox">
      <ul>
        <li><a href="#">首頁</a></li>
        <li><a href="#">關於我們</a></li>
        <li><a href="#">服務項目</a></li>
        <li><a href="#">聯絡我們</a></li>
      </ul>
    </div>
  </div>
  
  <div id="banner">
    <img src="banner.jpg" alt="橫幅圖片">
  </div>
  
  <div id="content">
    <div id="left">左欄內容</div>
    <div id="right">右欄內容</div>
  </div>
  
  <div id="footer">
    <p>版權資訊</p>
  </div>
</div>

CSS 樣式：

#wrap {
  width: 960px;
  margin: 0 auto;
}

#header {
  height: 100px;
  background: #333;
  color: white;
}

#banner {
  width: 100%;
  overflow: hidden;
}

#banner img {
  width: 100%;
}

#content {
  display: flex;
  min-height: 400px;
}

#left {
  width: 200px;
  background: #f0f0f0;
  padding: 20px;
}

#right {
  flex: 1;
  background: white;
  padding: 20px;
}

#footer {
  background: #666;
  color: white;
  text-align: center;
  padding: 20px;
}
